# Overview
This app is a blogging app for students of full stack bootcamps.  It's a place to rant or share your successes, or stash your dad jokes for later.

# Technologies Used
* HTML/CSS
* TailwindCSS
* Javascript
* Node.js
* Express
* Mongo/Mongoose
* Heroku
* Mailchimp API
